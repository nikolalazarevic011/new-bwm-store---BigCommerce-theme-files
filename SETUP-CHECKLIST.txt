1. ⬜ Get BigCommerce Store API Token

   Instructions:
   - Log in to BigCommerce admin panel
   - Go to: Settings → API → Store-level API Accounts
   - Click "Create API Account"
   - Set scopes:
     ✓ Themes: modify
     ✓ Store Information: read-only
   - Save and copy the Access Token (you won't see it again!)

2. ⬜ Configure Stencil CLI

   Open Command Prompt and run:

   nvm use 20.11.1
   cd "C:\Nikola\ME\New BWM Store\theme code"
   stencil init

   Enter when prompted:
   - Store URL: https://yourstore.mybigcommerce.com
   - Access Token: [paste the token from step 1]
   - Port: 3000 (or press Enter for default)

3. ⬜ Start Development Server

   Double-click: start-dev.bat

   OR run manually:

   nvm use 20.11.1
   stencil start

4. ⬜ Open Browser

   Navigate to: http://localhost:3000

   You should see your BigCommerce store theme running locally!
